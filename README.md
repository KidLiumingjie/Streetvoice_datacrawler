# Streetvoice_datacrawler
Get data from streetvoice

Run on Google Colab
### 開啟Streetvoice榜單 (根據榜單不同 你得到的連結也不同)

按下F12 進入開發者工具

Step 1.點選Network

Step 2.訂閱歌單 需先登入帳號

Step 3.右鍵進入連結
